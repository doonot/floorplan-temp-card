    Mehrfachauswahl

    Name/Entity übernimmt der erste Raum mit Sensor (sonst der zuerst gewählte).

    Hilfe

    1. „🖼 Plan-Bild laden…": Architektenplan als Zeichenvorlage laden (bleibt lokal, wird nie exportiert). Bild oder Projekt-JSON kann auch per Drag & Drop aufs Fenster gezogen werden.
    2. Mit ▭ Räume über die Zimmer ziehen (Raster 5 px).
    3. Raum anklicken → Name + Sensor-Entity zuweisen.
    3b. L-/T-förmige Räume: mehrere Rechtecke zeichnen, mit Shift-Klick zusammen auswählen → „Zu einer Fläche vereinen". Trennen jederzeit über „Flächen trennen".
    3d. 🌳/🌿/🪴: Bäume, Hecken (ziehen, auch schräg) und Pflanzen als Deko — ohne Entity, Größe im Panel. Schräge Raumformen: ⬠ Polygon-Werkzeug. Cmd/Ctrl-A: alles markieren, dann ziehen verschiebt das ganze Stockwerk (z. B. um Platz für den Garten zu schaffen — Canvas vorher über Breite/Höhe vergrößern).
    3c. 💡/🔌: Lichter und Steckdosen aufs Bild setzen (L), Entity zuweisen. ▤ Store (S): am Fenster entlang aufziehen — wird ein Balken, Endpunkte ziehen = Länge ändern. In der Karte: Licht-Tap schaltet, lang drücken = Details; Store-Tap öffnet die Steuerung, Füllung = wie weit unten.
    4. „YAML exportieren" → Config in HA einfügen (custom:floorplan-temp-card).
    Projekt wird automatisch im Browser gespeichert.
